# Medical RAG QA System

A Retrieval-Augmented Generation (RAG) system for medical question answering built with:
- TinyLlama 1.1B for inference
- FAISS for vector search
- Sentence Transformers for embeddings
- LangChain for RAG pipeline

## Features
- Fast and lightweight
- Local RAG pipeline
- Medical transcription dataset
- Clean interface with Streamlit
